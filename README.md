# hermit-deck-os

Bazzite is great, but they target such a wide user base that I often find it to be a little bloated. So, `hermit-deck-os` is a custom bootc image forked from `bazzite-deck-gnome` which strips out a lot of unnecessary things I don't like or ever use.

## Removals:

I find pre-installed applications to be really annoying, even if they don't ever use any resources, so I'm not a fan of keeping them.

Removals are as follows:

1. Waydroid
2. Numerous flatpaks

## Setups:

Some preferential automated setups that I like, including:

1. Sunshine
2. Decky
